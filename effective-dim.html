3D intuition: same features, different dimensionality

Six clouds in ℝ³ — same number of features, very different effective dimensionalities. Drag to rotate, scroll to zoom.

Key insight: all six clouds live in ℝ³ — same number of features, same number of columns in the data matrix. Effective dimensionality ranges from 1 to 3 depending entirely on how variance is distributed. A flat eigenspectrum → high ED. A spiky one → low ED.
